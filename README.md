# s1-Meal Helper
created by Yuminov Denis 11-109

Аналог большинства сайтов с рецептами, где пользователь может создавать свои рецепты.

Усложнение:

Персонализация: добавление функционала, позволяющего пользователям выбирать свои предпочтения.
Такой функционал может помочь пользователям быстрее найти рецепты, которые им подходят.

Красивый дизайн (но это не точно): приложение должно иметь привлекательный и интуитивно понятный дизайн,
который будет удобен для пользователей.

Возможность добавлять и отслеживать свои и чужие рецепты: дать возможность пользователям добавлять свои рецепты,
которые они хотят сохранить, а также отслеживать рецепты, которые они уже создали, пользовались.

Локализация: ru, английский 

Функция "Сохранить на позже": добавление функционала, который позволяет пользователям сохранять рецепты
на будущее, чтобы они могли вернуться к ним позже. Типа как черновик.
